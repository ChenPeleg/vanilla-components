Syntax Highlighter Color Themes

The syntax highlighter now supports three color themes inspired by popular code editors: bold (VS Code Dark+), calm (GitHub), and faded. Below are examples of each theme applied to the same code snippet.

Bold Theme (Default) - VS Code Dark+ Inspired

Based on the popular VS Code Dark+ theme with vibrant, professional colors.

Calm Theme - GitHub Inspired

Inspired by GitHub's syntax highlighting with balanced, comfortable colors.

Faded Theme

Subtle, muted colors using lighter shades (300-level).

Usage

To use a theme, simply add the theme attribute:

<highlighted-code theme="faded" code="const x = 5;"></highlighted-code>
